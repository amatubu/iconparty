IconParty
=========

 256 色のアイコンなどの画像に特化したペイントソフト兼アイコンエディタです。

動作環境
--------

 Mac OS X 10.6 以降で動作します。

 Mac OS X 10.9 / MacBook Air mid 2011 にて動作確認。

使い方
------

 オンラインマニュアルを参照のこと。

 http://amatubu.skr.jp/?Manual/IconParty

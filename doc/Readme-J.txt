// ソース公開について ////ソース公開の目的プログラムが複雑になり、また私の方もIconPartyの開発にかける時間があまりなくなってきたことから、開発の協力者を公につのろうというのが主な目的です。また、汚いソースですが、ひょっとしたらなにかの役にたつことがあるかも知れないという目的もあります。//ソースの２次使用について各ソースファイルの内容はフリーウェアとします。別のソフトウェア内で使用していただいて構いません。そのソフトのドキュメントのどこかにでも「IconPartyのソースファイルを参考にしました」とか書いていただけると非常にうれしいです。これらのソースファイルと一部不足しているファイルをあわせてコンパイルすると、公開されているIconPartyと同じものができあがりますが、それをそのまま公開することはご遠慮ください。なんらかの改良を加えた上で改良内容を私の方まで確認の上、改良後のソースもあわせて公開することを条件に公開を許可します。ただし、一般公開する場合にはアプリケーションのアイコン、名前、クリエータコードをIconPartyと別のものにしてください。また、改良後のソースをさらに別な人が改良することも可能にしてください。もちろん、ソフトのドキュメントには「IconPartyのソースを元に改良を加えたものです」というような一文を書いておいてください。個人で使用するかぎりはどのように使用していただいても構いません。//ソースを改良する時はオリジナルのソースファイルに改良を加える場合には、変更した部分、およびそのソースファイルの一番上に変更した人の名前、および日付を書いておいてください。できれば変更の内容も書いておいていただけるといいかと思います。ただし、個人で使用する場合は特に書く必要はありません。私自身もほとんど書いていないですし（汗）。//リソースの２次使用についてアイコン以外はご自由に使用していただいて構いません。---	00.6.4	naoki iimura
